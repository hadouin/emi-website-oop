# EMI - The APP that cares for your workforce
Repo pour l'APP-G10D

## ⚡️ Getting Started

Clone the project

```bash
git clone https://github.com/hadouin/emi-website-oop.git
```

Navigate to the project directory

```bash
cd emi-website-oop
```

Start the services
```bash
docker-compose up -d
```
